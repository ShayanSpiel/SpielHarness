# X Thread Template

## Frontmatter
title, created, platform: x, format: thread, status: draft, source, reader, pain, belief, point, meaning, proof

## Body structure
- Thread starter (tweet 1): Hook that makes them tap "show more"
- Tweets 2-4: The narrative — pain, belief, contradiction
- Tweets 5-7: The proof — evidence with specifics
- Tweet 8: The meaning — aha, first-person
- Tweet 9: The CTA — question or invitation
- Tweet 10: Summary pin with key takeaway

## Rules
- Each tweet: max 280 chars, self-contained, makes sense alone
- Number tweets: 1/10, 2/10...
- End each tweet with a reason to read the next one
- Last tweet is a pin with key takeaway and CTA
