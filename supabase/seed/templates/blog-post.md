# Blog Post Template

## Frontmatter
title, created, platform: blog, status: draft, source, reader, pain, belief, point, meaning, proof

## Body
- Hook / lede
- Context / the old belief
- The contradiction / the new point
- Proof: 3 concrete examples
- Meaning / what this means for the reader
- CTA

Max 2500 words.

## Notes
- Sections with subheadings.
- Proof with numbers and specifics.
