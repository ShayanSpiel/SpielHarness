# Web Search

Represents manual public web research using free sources.

## Free sources
- Search engine results
- Company websites
- Public docs
- Platform ad libraries
- Google Trends
- Public social profiles

## Rules
- Treat web results as unverified until a primary source confirms them.
- Record the URL and retrieval date when used.
- Do not claim private or paid data access.

## Output
Source list with short notes.
