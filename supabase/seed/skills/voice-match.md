# Voice Match

Loads and matches the writer's voice to the closest canonical example.

## Steps
1. Read voice.md, methodology.md, and examples.md from strategy knowledge
2. Identify which example's rhythm matches the current brief's Belief→Point shape
3. Instruct the writer to match that example's rhythm, sentence structure, and pacing
4. Flag any draft lines that drift from voice

## Output
Voice-matched drafting instructions and drift warnings.
