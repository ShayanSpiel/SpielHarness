# DuckDuckGo Search

Free, private web search with no API key required. Uses the DuckDuckGo API.

## When to use
- Find market signals, competitor intel, audience research
- SEO keyword research and search intent analysis
- Validate claims with public sources
- Stay current with industry news and trends

## Rules
- Treat web results as unverified until a primary source confirms them
- Record the URL and retrieval date when used
- Prefer recent results (last 12 months) for market research
- Do not claim private or paid data access

## Output
Source list with title, URL, snippet, and key takeaway.
