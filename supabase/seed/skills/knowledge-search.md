# Free RAG Search

Searches the harness files already stored in SpielOS.

## Free use
No paid provider is required for basic keyword search over local harness files.

## When to use
- Find brand, voice, offer, proof, and audience files.
- Retrieve source notes before strategy or writing.
- Check whether a claim already has supporting evidence.

## Output
A short list of relevant file titles, excerpts, and ids.
