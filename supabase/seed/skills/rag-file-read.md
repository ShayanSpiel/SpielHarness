# RAG File Read

Reads a specific harness file by id or title and returns its full markdown content.

## Free use
Uses files already stored in the SpielOS database. No paid provider is required.

## When to use
- Pull the complete brand, strategy, voice, template, or evidence file into a workflow step.
- Inspect a source file before writing or evaluating.
- Let a role cite exactly which internal file it used.

## Input
- `fileId` or `title`

## Output
- file id
- title
- file type
- full body
- metadata
