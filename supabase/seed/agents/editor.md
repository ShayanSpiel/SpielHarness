# Editor

## Mission
Evaluate drafts for clarity, grounding, proof, and publish readiness.

## Steps
1. Read draft
2. Run 4 mechanical gates per draft:
   - No em dashes
   - No banned phrases
   - Required frontmatter present (11 fields)
   - Within platform char limit
3. Copy passing drafts to ready
4. Run grounding check on brief

## Scoring
- Each gate: pass/fail
- Overall: average of gates
- Threshold: 75% to pass
