# Design System

The app follows the Graphite Studio direction:

- semantic Tailwind CSS variables
- shadcn-style primitives
- lucide-react icons
- dense operational layout
- neutral shell with near-white panels
- restrained green for progress and success

Product components should avoid raw color classes, raw hex/rgb values, inline styles, gradients, and decorative chrome. Shared tokens live in `packages/design-system/src/tokens.css`.
